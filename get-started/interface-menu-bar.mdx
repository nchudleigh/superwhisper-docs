## Overview

The Superwhisper menu bar icon is a crucial interface element that provides real-time status updates and quick access to key functions.

---
## Status Indicator Dot

The menu bar icon features a color-coded dot that indicates the current status of Superwhisper:

<CardGroup cols={2}>
  <Card title="Yellow" icon="circle" horizontal>
    Model loading in progress
  </Card>
  <Card title="Red" icon="circle" horizontal>
    Active recording in progress
  </Card>
  <Card title="Blue" icon="circle" horizontal>
    Processing dictation
  </Card>
  <Card title="Green" icon="circle" horizontal>
    Processing completed
  </Card>
</CardGroup>

---
## Menu Bar Options

<Steps>
  <Step title="Recording Controls">
    - Start/Stop recording
  </Step>
  <Step title="File Management & Transcription">
    - Transcribe external files
    - Access Superwhisper history
  </Step>
  <Step title="Application Controls">
    - Open main settings window
    - Change active mode
    - Check for updates
    - Quit application
  </Step>
</Steps>

---
## Interaction Options

The Superwhisper menu bar icon provides two main interaction modes:

<Steps>
  <Step title="Default Behavior">
    Clicking the icon opens the menu with all available options
  </Step>
  <Step title="Quick Recording">
    You can enable this in the [advanced settings](settings-advanced#interface-customization). This setting changes the default behavior to:
    - Left click toggles recording
    - Right click opens the full menu
  </Step>
</Steps>
