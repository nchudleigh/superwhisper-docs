## Key Features
**Notes Mode transforms your spoken words by:**
- Preserving original context while improving clarity
- Formatting text for optimal readability
- Highlighting key points and action items
- Adapting content structure based on topic and key topics

<Tip>Note Mode is perfect for lectures, meetings, brainstorming sessions, and quick thought capture.</Tip>
